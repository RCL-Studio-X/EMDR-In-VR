# External Assets & Packages

These are NOT tracked in git. Re-import on any new machine after pulling.

| Asset | Source | Folder |
|-------|--------|--------|
| WOC City/Park Environment | Unity Asset Store | `Assets/WOC/` |
| 12 Animated Butterflies Pack | Unity Asset Store | `Assets/12 Animated butterflies pack/` |
| Modern UI Pack (MUIP) | Unity Asset Store | `Assets/Modern UI Pack/` |
| XR Interaction Toolkit Samples | Unity Package Manager | `Assets/Samples/XR Interaction Toolkit/` |
| XR Hands Samples | Unity Package Manager | `Assets/Samples/XR Hands/` |

## How to re-import
1. **Asset Store packages** → Unity Package Manager → My Assets → find and import
2. **Package Manager samples** → Window → Package Manager → find the package → Samples tab → Import
